## 特性

- 强大的主题定制功能，实时动态切换
- 多页签功能，助力后台管理高效开发
- 内置权限控制，兼容 ABAC 和 RBAC 模型
- 丰富的内置业务组件和常用页面模板

## 使用

### 拉取代码

```bash
git clone https://github.com/stepui/stepin-template.git
```

### 安装依赖

```bash
yarn install
```

### 启动

```
yarn dev
```

### 预览

启动成功后，控制台会显示本地访问地址：http://127.0.0.1:5173，浏览器打开即可预览。

更新多信息请参考 [使用文档](http://stepui.gitee.io/stepin-template-docs/)
